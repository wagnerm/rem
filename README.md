# rem-cli
Remember things, on the CLI

## Installation

```
$ cargo install rem-cli
```

Or clone this repository and compile!

## Usage

```
$ rem-cli add Is mayonnaise an instrument?

$ rem-cli add 987654

$ rem-cli cat
Is mayonnaise an instrument?
987654
```
